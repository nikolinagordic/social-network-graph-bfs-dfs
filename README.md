# social-network-graph-bfs-dfs

# Primjena grafova u modelovanju društvenih mreža

Projekat iz predmeta Strukture podataka.

## Opis

U projektu je implementirana društvena mreža korišćenjem strukture podataka graf.

Implementirane funkcionalnosti:

- Struktura grafa pomoću liste susjedstva
- BFS (Breadth-First Search)
- DFS (Depth-First Search)
- Pronalaženje najkraćeg puta
- Analiza povezanih komponenti
- Analiza broja veza korisnika
- Preporuka prijatelja
- Vizualizacija mreže pomoću NetworkX i Matplotlib biblioteka

## Tehnologije

- Python
- Google Colab
- NetworkX
- Matplotlib
